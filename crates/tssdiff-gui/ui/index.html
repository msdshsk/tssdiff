リポジトリを開いてください

Ctrl+O、または右上のフォルダボタンから git リポジトリを選択します。
コマンドラインからは tssdiff-gui <path> で直接開けます。

設定

エディタ

「エディタで開く」で使うコマンド。空にして保存するとリセットされ、次回に選択ダイアログが出ます。

Git バックエンド

auto = git コマンドがあれば CLI、無ければ内蔵 gix(git 不要)。変更はリポジトリを開き直すと反映されます。

エージェント送信先 (sink)

コメント/質問の送信方法。command は標準入力に JSON payload を渡します。

保存先: ~/.config/tssdiff/config.yaml(TUI 版と共有)

エディタの設定

「エディタで開く」で使うエディタを選択してください。設定は config.yaml の editor に保存され、TUI 版と共有されます。
